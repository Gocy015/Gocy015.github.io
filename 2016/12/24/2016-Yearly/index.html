2016年度总结

发表于 2016-12-24 更新于 2017-05-27

转眼便来到了 2016 年的最后一周。不得不说，2016 对我而言是充满变化的一年，在这圣诞之际，写一篇博客记录一下即将过去的这一年。

技术

吃饭的本事，还是不能落下。

工作

如果要给今年发生的大事排个榜，那么工作必然是榜上头名了。

自三月份开始实习到现在，竟也有小一年时间了。最深刻的体会便是，你在学校学到的那些，来到企业内根本就不够用。在学院跟着老师开发的时候，版本管理基本靠手动压缩成 zip ，从没试过看大段大段的别人的代码，而由于项目代码简单，功能的增删改也非常简单。直到进入项目组，看到 1 个多 G 的项目工程，我才知道什么叫“项目”（虽然很大一部分空间用来存各种 UI 的切图）。我第一次看一个具有完整生态的应用：账号登录和信息存储、获取 token 登入频道、与服务器做心跳同步等等，前后花了近两周才走通整个项目的主体代码。对了，把 svn 搞明白都花了我大半天时间。

阅读项目登录模块时所做的笔记

之前以为自己多牛逼，进来才发现自己是菜鸡。当然，仔细的阅读也为我接下来改 bug 并开始实现需求打下了良好的基础，基本要依赖什么模块都不太需要问别人，自己试试就能做得七七八八。但这个这第一个项目给我提升最多的或许不是编码硬能力的提高，而是如何进行团队开发这种软实力的提高。如何研读文档、与后台定接口联调、正确的进行版本控制、和测试共同定位 bug 等等，这些都是我在之前的项目开发中缺失的技能。

入职以来被提过的 Bug

积累

回头再看，好像很难说开发哪项任务让我的编码能力有了怎样的提高，但不论是由于需求开发的需要，还是源于团队分享，抑或是自己找到的一些技术文章，今年所做的技术积累也非常可观。

我也有开源项目了

终于在 我的Github 有了持续地产出。今年写了几个通用的 UI 组件（ PieChart , ExpandableTableView , GuidanceView ），功能不复杂，但每一个组件在编写的过程中，我都尽量做更高层次的抽象，站在“第三方使用者”的角度考虑，如何写出更易用，更“黑盒”的库。尽管几个组件库加起来还不到十个 star ，但不论如何，也是迈开了在社区共享代码的第一步了（ 为啥我写的最辛苦的 Core Animation 动画库 偏偏没人看呢）。

Community Debugging

我也终于迈出了阅读优秀源码的一步。今年我阅读了 RunLoop 相关的部分 CoreFoundation 源码以及 AsyncDisplayKit 的部分源码，并且也记录下了相应的感想（RunLoop源码阅读笔记 、AsyncDisplayKit源码阅读笔记）。阅读这些代码对我的影响是相对隐式的，尽管在简单地功能开发中难以体现，但现在无论是 debug ，还是开发中对所负责模块的解耦，我都感觉自己的“编程思维”比起之前有了很大的提高。另外，我还加入了 掘金翻译计划 ，成为了一名技术文章的译者。我所学到的几乎一切编程知识都来源于开源社区，现在，我也希望通过自己的绵薄之力，回馈社区。

Github + Hexo

还有一个不得不提的就是这个博客了。给我 大楷哥 一万个感谢，不仅仅因为教我搭博客、学 JavaScript ；正是因为看到你，一个在我身边的人，有着这么多优秀的习惯，我才下决心开始记录这学习之路的点点滴滴。

Dear JavaScript

“微信小程序革客户端开发的命”风波虽然已经过去，但 JavaScript 的大浪可还正在汹涌翻滚。在 Web 前端，JavaScript 优秀框架不断涌现。在 Native 客户端，混编甚至纯 JavaScript 实现层出不穷。在后端，LeanCloud 提供 Node.js 实现数据存储逻辑。正如我们项目组组长在技术分享上说的那样，“在未来，能用到 JavaScript 的地方，最终都会用上 JavaScript ”，我还并不了解这门语言到底有多强大，而又为什么会这么强大。但我已起步，开始了解并学习它。

其它

终于自己动手写了单元测试、学会了利用 Charles / WireShark 抓包、终于搞懂了 http 包的结构等等。更重要的是，已经初步养成了学习的惯性，成功染上了不做点技术积累就不舒服的毛病。

我的 Github Commit

生活

我常常想，为什么我为我，与他人不同。别人要求你做的事，你不做，任何人都可以代替你做。唯独那些你选择做的事，让你成为了自己。

初入社会，我最担心的一件事就是加班。并不是怕累或是什么，而是害怕失去了自我。在许多事情上，我和很多同龄人一样，非常的迷茫，但我很庆幸，打心底里，我知道自己想坚持什么。

我先为我，后为一个程序员。

健身

很高兴，在入职的第一年，我成功地保持了健身的习惯。有的人把下班后的时间用来加班，他们或许能够获得更好地绩效。有的人把下班后的时间用来积累，他们或许最后能够成为行业精英。我习惯用举铁结束忙碌的一天，我也喜欢看着自己慢慢变成自己想象中的那样的人。

摄于 2016.9.20

游戏

小时候，就常和爸爸玩 PS2 ，从没想过有一天，我会真的买下属于自己的游戏机。那一刻我才觉得自己真的长大了，可以通过自己的努力来满足自己的渴望。

然而我已经把 2K17 卖了
